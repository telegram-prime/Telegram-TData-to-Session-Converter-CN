# TD2S Converter – Telegram **TDATA → SESSION**（或 SESSION+JSON）  
将 Telegram Desktop 的 TDATA 格式账号转换为 Telegram 会话文件（SESSION 或 SESSION+JSON）的工具。

## ⚙️ 主要功能

| 功能 | 说明 |
|------|------|
| **TDATA → SESSION / SESSION+JSON** | 将 Telegram Desktop 的 TDATA 账号转换为 SESSION 或 SESSION+JSON。 |
| **多线程操作** | 并行处理多个文件，提升批量转换速度。 |
| **冷启动转换** | 可在不连接账号的情况下将 SESSION+JSON 转换为 TDATA。 |
| **旧层兼容** | 选项可保持旧 Telegram 层或保存为新层。 |
| **新会话创建** | 可为账号生成新的会话或保存当前会话。 |
| **TDATA 再授权** | 将 TDATA 用于新账户的重新授权（即导出 TDATA）。 |
| **自定义重试 / 延迟** | 设置连接尝试次数及两次尝试间的等待时间。 |
| **禁用其他会话** | 在转换或重新授权期间可断开账号所有其它活跃会话。 |
| **设备仿真配置** | 灵活配置用于连接会话的设备仿真方式。 |
| **代理支持** | 为每一次连接使用唯一 HTTP 或 SOCKS5 代理（支持大量代理）。 |
| **跳过无效会话** | 若无法连接则直接跳过。 |
| **结果分类** | 有效账号 → “valid”，无效账号 → “invalid” 文件夹。 |
| **日志保存** | 可选地保存操作日志。 |
| **备份功能** | 在转换前创建源文件的备份。 |
